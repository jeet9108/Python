# AuditPro 📊

**AuditPro** is a professional-grade desktop application designed for Chartered Accountants (CAs) and audit firms to streamline the management and review of working papers. Built with Python and MySQL, it provides a secure and organized environment for audit workflows.

## ✨ Key Features

- **Dynamic Dashboard**: A high-level overview showing pending and completed reviews at a glance.
- **Working Papers Management**: Track, review, and update the status of audit documents (Accepted, Rejected, Pending).
- **User Management**: Admin-level control for managing team access and roles.
- **Database Manager**: Seamless integration with MySQL for reliable data storage and retrieval.
- **Modern UI/UX**: A clean, intuitive interface built with Tkinter, featuring a dark-themed sidebar and card-based statistics.

## 🛠️ Technology Stack

- **Language**: Python 3.x
- **GUI Framework**: Tkinter
- **Database**: MySQL
- **Libraries**: `mysql-connector-python`, `os`, `sys`

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed
- MySQL Server running locally
- Required Python packages:
  ```bash
  pip install mysql-connector-python
  ```

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jeet9108/Python.git
   ```
2. Import the database schema (if available) or ensure a database named `ca` is created in MySQL.
3. Run the application:
   ```bash
   python Codes/modules/Login.py
   ```
